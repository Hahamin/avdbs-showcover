# ShowCover — avdbs 표지 복원 크롬 확장 프로그램

avdbs.com에서 표시되지 않는 AV 작품 표지 이미지를 자동으로 복원해주는 Chrome 확장 프로그램입니다.

---

## 주요 기능

- 🖼️ 작품 목록 및 상세 페이지의 표지 이미지 자동 복원
- 🔍 품번 옆 **[🖼️]** 버튼으로 표지 팝업 즉시 확인
- 📋 **[품번복사]** 버튼으로 클립보드에 한 번에 복사
- 🎬 상세 페이지 미리보기 버튼 지원
- ⚡ 별도 조작 없이 페이지 접속만 해도 자동 실행

---

## 설치 방법

### 1. 다운로드

아래 **Releases** 페이지에서 최신 버전의 zip 파일을 받으세요.

👉 [Releases 페이지](https://github.com/Hahamin/avdbs-showcover/releases/latest)

### 2. 설치

1. 다운로드한 zip 파일의 압축을 해제합니다.
2. Chrome 주소창에 `chrome://extensions` 입력
3. 우측 상단 **개발자 모드** 토글 켜기 (ON)
4. **압축 해제된 확장 프로그램을 로드합니다** 클릭
5. 압축 해제한 폴더 선택

> 자세한 설치 방법은 폴더 안의 `적용_가이드.txt` 를 참고하세요.

---

## 사용 방법

[avdbs.com](https://www.avdbs.com) 에 접속하면 자동으로 실행됩니다.

| 버튼 | 기능 |
|------|------|
| 🖼️ | 해당 품번의 표지 팝업 표시 |
| 품번복사 | 품번을 클립보드에 복사 |

팝업은 바깥 클릭 또는 우측 상단 ✕ 버튼으로 닫을 수 있습니다.
