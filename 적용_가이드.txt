================================================================
  ShowCover - Chrome 확장 프로그램 설치 및 적용 가이드
================================================================

[ 소개 ]
ShowCover는 avdbs.com에서 표시되지 않는 AV 작품 표지 이미지를
자동으로 복원해 주는 Chrome 확장 프로그램입니다.


----------------------------------------------------------------
  1단계. 설치 준비
----------------------------------------------------------------

1. Google Chrome 브라우저가 설치되어 있어야 합니다.
   (Chromium 계열 브라우저 - Whale, Edge 등도 가능)

2. ShowCover 폴더를 PC의 원하는 위치에 저장합니다.
   예) C:\Users\사용자명\Documents\ShowCover

   ※ 설치 후 폴더를 이동하거나 삭제하면 확장 프로그램이
      작동하지 않으므로 고정된 위치에 저장하세요.


----------------------------------------------------------------
  2단계. Chrome 확장 프로그램 로드
----------------------------------------------------------------

1. Chrome 브라우저를 열고 주소창에 아래 주소를 입력합니다.

     chrome://extensions

2. 페이지 우측 상단의 [개발자 모드] 토글을 켭니다 (ON).

3. 좌측 상단에 나타나는 [압축 해제된 확장 프로그램을 로드합니다]
   버튼을 클릭합니다.

4. ShowCover 폴더를 선택하고 [폴더 선택]을 클릭합니다.

5. 확장 프로그램 목록에 "ShowCover"가 나타나면 설치 완료입니다.


----------------------------------------------------------------
  3단계. 사용 방법
----------------------------------------------------------------

1. avdbs.com 에 접속합니다.
   https://www.avdbs.com

2. 작품 목록 또는 상세 페이지에서 표지 이미지가 자동으로
   복원됩니다. (별도 조작 없이 자동 실행)

3. 각 품번 옆에 나타나는 [🖼️] 버튼을 클릭하면 팝업으로
   해당 작품의 표지 이미지를 바로 확인할 수 있습니다.


----------------------------------------------------------------
  4단계. 업데이트 방법
----------------------------------------------------------------

1. 기존 ShowCover 폴더 내용을 새 버전의 파일로 교체합니다.

2. chrome://extensions 페이지에서 ShowCover 카드의
   새로고침(↺) 버튼을 클릭합니다.

3. avdbs.com 페이지를 새로고침하면 업데이트가 적용됩니다.


----------------------------------------------------------------
  주의사항
----------------------------------------------------------------

- 이 확장 프로그램은 avdbs.com 에서만 작동합니다.
- Chrome 웹 스토어를 통한 배포 버전이 아니므로, Chrome을
  업데이트하거나 재시작할 때 확장 프로그램 비활성화 경고가
  뜰 수 있습니다. 이 경우 chrome://extensions 에서 다시
  활성화해 주세요.
